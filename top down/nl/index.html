Top Down Shooter

Shine Project Top down shooter T-Rex game

Dit is een van mijn projecten waar ik het meest trots op ben. Het ziet er leuk uit en werkt goed. Ik heb er ook veel tijd in gestoken, het is jammer dat ik nooit de kans kreeg om het af te maken. Dat komt omdat ik dit moest maken tijdens het shine project. Ik heb geleerd dat werken aan twee projecten tegelijk moeilijk is en dat ik het beter moest plannen. Het project zelf verliep vlot.

Ik gebruikte Unity als engine en gebruikte C# om alles te coderen. Ik heb de speler kleine vierkantjes laten schieten die bij impact vernietigen. De zombies gaan naar de speler toe en als ze botsen doet dat ongeveer 5-10 schade aan de speler.
