T-Rex Game

Shine Project Top down shooter T-Rex game

Dit is een van mijn eerste projecten die ik gemaakt heb. Ik heb een tutorial gebruikt om het te maken, maar nadat ik het spel had gemaakt heb ik ook een menuscherm en een helpscherm gemaakt om het wat origineler te maken. Ik heb er een paar uur aan besteed, het is niet zo lang maar het is een van de eerste projecten die ik maakte dus toen was ik trots toen het werkte. Ik leerde dat ik mezelf moest uitdagen en proberen origineler te zijn.

Ik heb dit gemaakt met Visual Studio. Ik probeerde het origineler te maken door een menuscherm en helpscherm te maken. Ik heb ook gemaakt dat als je dood gaat, je het spel opnieuw kunt starten.
