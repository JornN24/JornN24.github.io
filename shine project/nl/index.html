Shine Project

Shine Project Top down shooter T-Rex game

Ik heb voor dit project gekozen omdat het een van mijn beste projecten is die ik gemaakt heb. Ik heb er veel tijd in gestoken, ik ben ook erg trots op mezelf dat ik dit gemaakt heb. Ik had veel obstakels, maar ik heb ze kunnen oplossen. Ik heb geleerd mijn projecten niet te overhaasten en een goede to do lijst te maken. Ook heb ik geleerd door te gaan en niet op te geven als je het probleem of de bug niet kunt vinden.

Ik gebruikte unity als engine en gebruikte C# om alles te coderen. Ik heb ook de terrain tool gebruikt om het landschap te vormen waar de speler op kan spelen. De gebouwen en vijanden zijn assets uit de asset store. Ik heb ook een startscherm met instellingen gemaakt met behulp van een andere scene.
